# Day 9 - Wi-Fi Web Server LED Control Using ESP32

## 📅 Date

16 July 2026

## 📌 Project Overview

This project demonstrates how to create a web server using the ESP32. The ESP32 connects to a Wi-Fi network and hosts a web page that allows users to turn an LED ON and OFF from any device connected to the same network.

## 🎯 Objective

* Learn how to connect the ESP32 to a Wi-Fi network
* Create a simple HTTP web server
* Control an LED through a web browser
* Understand basic client-server communication

## 🛠️ Hardware

* ESP32 Development Board
* LED
* 220Ω Resistor
* Breadboard
* Jumper Wires
* USB Cable

## 💻 Software

* Arduino IDE
* Arduino Core for ESP32

## 🛠️ Technologies Used

* ESP32 Development Board
* C++
* Arduino Framework
* Wi-Fi
* HTTP Web Server
* HTML
* GPIO Output
* Embedded Systems
* Git
* GitHub

## 📖 What I Learned

* Connecting the ESP32 to a Wi-Fi network
* Creating an HTTP server
* Handling client requests
* Controlling hardware using a web interface
* Combining networking with embedded systems
