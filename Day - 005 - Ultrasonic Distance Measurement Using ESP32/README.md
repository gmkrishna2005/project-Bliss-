# Day 5 - Ultrasonic Distance Measurement Using ESP32

## 📅 Date

12 July 2026

## 📌 Project Overview

This project demonstrates how to measure the distance between an object and an ESP32 using the HC-SR04 Ultrasonic Sensor. The sensor emits ultrasonic waves and calculates the distance by measuring the time taken for the echo to return.

## 🎯 Objective

* Learn how ultrasonic sensors work
* Interface the HC-SR04 with the ESP32
* Measure the distance to nearby objects
* Display distance values on the Serial Monitor

## 🛠️ Hardware

* ESP32 Development Board
* HC-SR04 Ultrasonic Sensor
* Breadboard
* Jumper Wires
* USB Cable

## 💻 Software

## 🛠️ Technologies Used

* ESP32 Development Board

## 📖 What I Learned

* Working principle of ultrasonic sensors
* Generating trigger pulses
* Measuring echo pulse duration
* Calculating distance using the speed of sound
* Interfacing digital sensors with ESP32
