# Day 8 - Display Ultrasonic Sensor Data on OLED Using ESP32

## 📅 Date

15 July 2026

## 📌 Project Overview

This project displays real-time distance measurements from an HC-SR04 Ultrasonic Sensor on a 0.96-inch SSD1306 OLED display using I²C communication. Instead of viewing the readings on the Serial Monitor, the measured distance is shown directly on the OLED, creating a more user-friendly embedded application.

## 🎯 Objective

* Learn I²C communication
* Interface an SSD1306 OLED with the ESP32
* Display real-time sensor data
* Combine multiple peripherals in one project
