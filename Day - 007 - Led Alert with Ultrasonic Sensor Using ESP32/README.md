# Day 7 - led alert alert with Ultrasonic Sensor Using ESP32

## 📅 Date

14 July 2026

## 📌 Project Overview

This project combines an HC-SR04 Ultrasonic Sensor with an led to create a simple proximity alarm. The ESP32 continuously measures the distance to nearby objects and activates the led when an object is detected within a predefined threshold.

## 🎯 Objective

* Interface multiple peripherals with the ESP32
* Measure distance using the HC-SR04 sensor
* Activate a led based on sensor input
* Learn simple event-driven programming
