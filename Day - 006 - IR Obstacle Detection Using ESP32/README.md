# Day 6 - IR Obstacle Detection Using ESP32

## 📅 Date

13 July 2026

## 📌 Project Overview

This project demonstrates how to interface an IR Obstacle Sensor with an ESP32 to detect nearby objects. The sensor emits infrared light and detects its reflection to determine whether an object is present. Based on the sensor output, an LED is turned ON or OFF.

## 🎯 Objective

* Learn how an IR obstacle sensor works
* Interface a digital sensor with the ESP32
* Read digital input using `digitalRead()`
* Control an LED based on obstacle detection
