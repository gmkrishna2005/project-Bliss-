# Day 2 - Push Button LED Control

## 📅 Date

09 July 2026

## 📌 Project Overview

In this project, an ESP32 reads the state of a push button and controls an LED based on the button press. This project introduces digital input handling and demonstrates how to use the ESP32's internal pull-up resistor. It also covers the basics of button debouncing to prevent false or multiple button detections.

## 🎯 Objective

* Learn how to read digital inputs
* Understand internal pull-up and pull-down resistors
* Control an LED using a push button
