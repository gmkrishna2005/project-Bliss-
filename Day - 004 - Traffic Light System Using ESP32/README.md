# Day 4 - Traffic Light System Using ESP32

## 📅 Date

11 July 2026

## 📌 Project Overview

This project simulates a basic traffic light system using three LEDs connected to an ESP32. The LEDs represent the red, yellow, and green traffic signals and operate in the same sequence as a real-world traffic light.

## 🎯 Objective

* Learn to control multiple GPIO outputs
* Simulate a real-world traffic signal
* Understand sequential programming and timing
* Build the foundation for finite state machine (FSM) concepts

## 🛠️ Hardware

* ESP32 Development Board
* Red LED
* Yellow LED
* Green LED
* 3 × 220Ω Resistors
* Breadboard
* Jumper Wires
* USB Cable

## 💻 Software

* Arduino IDE
* Arduino Core for ESP32

## 🛠️ Technologies Used

* ESP32 Development Board
* C++
* Arduino Framework
* GPIO Output
* Embedded Systems
* Git
* GitHub

## 📖 What I Learned

* Configuring and controlling multiple GPIO pins
* Implementing sequential logic
* Simulating a real-world embedded application
* Understanding timing using `delay()`
