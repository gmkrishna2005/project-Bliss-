# Day 3 - Control LED Brightness Using PWM

## 📅 Date

10 July 2026

## 📌 Project Overview

This project demonstrates how to control the brightness of an LED using Pulse Width Modulation (PWM) on the ESP32. Instead of simply turning the LED ON or OFF, PWM rapidly switches the LED on and off at a high frequency, allowing different brightness levels by changing the duty cycle.

## 🎯 Objective

* Learn the fundamentals of Pulse Width Modulation (PWM)
* Control LED brightness by varying the duty cycle
* Understand PWM frequency and resolution on the ESP32
* Generate smooth brightness transitions
